🔧 安装方式
在 MoviePilot 后台 → 插件市场 → 添加第三方插件仓库：

https://github.com/zhuzhug/MoviePilot-Plugins
在插件列表中找到对应插件，点击安装即可。
🧑‍💻 开发说明
本仓库遵循 MoviePilot 第三方插件开发规范：

plugins/ 目录存放插件代码
package.json / package.v2.json 为插件清单
每个插件独立子目录，主类在 __init__.py 中编写
详细开发文档请参考 MoviePilot 官方插件市场。
