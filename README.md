# MoviePilot-Plugins

个人维护的 MoviePilot 第三方插件仓库。

## 订阅本仓库

在 MoviePilot 中打开 **设定 → 插件 → 插件市场**，添加以下地址：

```
https://github.com/zhuzhug/MoviePilot-Plugins
```

刷新后即可在插件市场中检索并安装本仓库提供的插件。

## 插件列表

### 辅种查看（CrossSeedView）

跨下载器识别辅种关系的可视化插件。以「种子名 + 文件大小」作为分组键，聚合当前所有下载器中的种子，帮助快速定位孤种、冗余种子与跨盘辅种情况，并可在开启安全开关后从界面直接删除单个辅种副本。

#### 主要功能

- **跨下载器辅种识别**：统一扫描所有已配置的下载器（qBittorrent / Transmission），按种子名与文件大小分组归并。
- **交互式筛选**：详情页顶部提供筛选表单，支持按下载器、名称关键词、保存路径关键词、辅种数量区间、文件大小区间进行过滤，条件持久化保存至插件配置。
- **可折叠分组卡片**：每个辅种组以可折叠面板展示，标题栏内联显示辅种数、单份体积、涉及的下载器；展开后列出全部辅种明细（下载器、保存路径、体积）。
- **可选删除**：在设置中显式开启「允许删除辅种（危险）」后，明细行提供「仅删种」与「删种 + 文件」两个操作；后端调用 MoviePilot 统一的 `remove_torrents` 通道，兼容各下载器差异。
- **定时与手动扫描**：默认按 CRON 定时全量扫描（`0 4 * * *`），详情页支持一键「立即重新扫描」。
- **性能与安全**：全量结果缓存在内存中，筛选与展示均在缓存上完成，避免频繁访问下载器；删除操作受安全开关保护，默认关闭。

#### 使用建议

- 首次使用建议保持「允许删除辅种」为关闭状态，先通过筛选熟悉本地辅种分布。
- 大量种子场景下（>10k），建议将 CRON 设为低峰时段，避免与站点刷流、同步任务冲突。
- 删除单个辅种前建议查看该分组内的做种数与来源下载器；**做种量越充足，删除单个副本对整体可用性的影响越小**。
- 云盘挂载路径下的种子建议通过路径关键词过滤后再进行清理，避免误删本地做种源。

#### 配置项

| 分组 | 参数 | 说明 |
| --- | --- | --- |
| 基础 | `cron` | 定时全量扫描表达式，默认 `0 4 * * *` |
| 基础 | `refresh_on_init` | MoviePilot 启动时是否立即执行一次扫描 |
| 基础 | `include_all_tags` | 是否包含无 MoviePilot 标签的种子 |
| 筛选 | `min_count` / `max_count` | 最小 / 最大辅种数（`max_count=0` 表示不限） |
| 筛选 | `downloader_filter` | 限定单个下载器 |
| 筛选 | `name_keyword` | 名称关键词，`\|` 分隔多个，不区分大小写 |
| 筛选 | `path_keyword` | 保存路径关键词，不区分大小写 |
| 筛选 | `size_min_gb` / `size_max_gb` | 单份体积区间（GB，`0` 表示不限） |
| 安全 | `allow_delete` | 是否在详情页显示删除按钮，默认关闭 |

#### 兼容性

- MoviePilot：v2 插件规范
- 下载器：qBittorrent、Transmission
- 页面渲染：Vuetify（内置 PageRender，无需前端联邦）
